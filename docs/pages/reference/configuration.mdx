h1: YAML Configuration Reference

Teleport uses the YAML file format for its configuration files.

Teleport [Roles](configuration/roles.mdx) and [Users](configuration/users.mdx) can also be configured using YAML.

Teleport [Plugins](configuration/plugins.mdx) are configured with TOML.
