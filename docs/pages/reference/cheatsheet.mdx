## Configure and start Teleport with Acme

```bash
# Configure Teleport with TLS certs
sudo teleport configure \
  --acme --acme-email=your_email@example.com \
  --cluster-name=tele.example.com \
  -o file

# Launch Teleport
sudo teleport start
```


