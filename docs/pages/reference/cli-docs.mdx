h1: Command Line (CLI) Reference
---

Teleport is made up of three CLI tools.

- [teleport](cli/teleport.mdx): The Teleport daemon that runs the Teleport Service, and acts as a daemon on a node allowing SSH connections.
- [tsh](cli/tsh.mdx): A tool that lets end users interact with Teleport nodes. This replaces `ssh`.
- [tctl](cli/tctl.mdx): An administrative tool that can configure Teleport Auth Service.
