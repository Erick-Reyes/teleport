## tsh db ls

Lists available databases and their connection information.

```shell
tsh db ls
```

Displays only the databases a user has access to (see [RBAC](../database-access/rbac.mdx)).

## tsh db login

Retrieves database credentials.

```shell
tsh db login example
tsh db login --db-user=postgres --db-name=postgres example
```

<table>
  <thead>
    <tr>
      <td>Flag</td>
      <td>Description</td>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>`--db-user`</td>
      <td>Optionally, set default database account name to connect as.</td>
    </tr>
    <tr>
      <td>`--db-name`</td>
      <td>Optionally, set default database name to connect to.</td>
    </tr>
  </tbody>
</table>

## tsh db logout

Removes database credentials.

```shell
tsh db logout example
tsh db logout
```

## tsh db env

Outputs environment variables for a particular database.

```shell
tsh db env
tsh db env --db=example
eval $(tsh db env)
```

<table>
  <thead>
    <tr>
      <td>Flag</td>
      <td>Description</td>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>`--db`</td>
      <td>Database server name to print environment for. Required, if logged into multiple databases simultaneously.</td>
    </tr>
  </tbody>
</table>

## tsh db config

Prints database connection information. Useful when configuring GUI clients.

```shell
tsh db config
tsh db config --db=example
```
