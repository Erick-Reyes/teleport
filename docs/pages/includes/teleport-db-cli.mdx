## teleport db start

Starts Teleport Database Service agent.

```bash
sudo teleport db start \
    --token=/path/to/token \
    --auth-server=proxy.example.com:3080 \
    --name=example \
    --protocol=postgres \
    --uri=postgres.example.com:5432
```
